# Portfolio V2

This is a remade version of the portfolio with a modern design, glassmorphism effects, and animations.

## Features
- **Modern UI**: Glassmorphism design with a clean layout.
- **Animations**: Scroll animations using AOS and a custom particle background.
- **Theme Support**: Dark and Light mode toggle.
- **Responsive**: Fully responsive for mobile and desktop.
- **Project Filtering**: Filter projects by category.

## How to Use
Simply open `index.html` in your browser.

## Customization
- **Images**: Replace images in the `img` folder.
- **Colors**: Edit the CSS variables in `css/style.css`.
- **Content**: Update the text in `index.html`.
